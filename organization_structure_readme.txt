Folder organization

/css - contains all css stylings for the project
/Documents - contains all documents and or files that are linked to in any html pages
/files - contains all files related to project work that is not linked to(wiring diagrams, assignment pages)
/Images - contains all images used for the website
/js- contains all javascript files
/old - contains older files that haven't been either deleted or organized
../ everything else is in the root folder (html/php)

Logbook files all need to become php files at some point. This will be on each individual.
